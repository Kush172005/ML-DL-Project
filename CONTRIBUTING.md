# Contributing to Hybrid Temporal Forecaster

Thank you for your interest in contributing to this project! As a final academic project, we welcome feedback and suggestions for Phase 2 extensions.

## How to Contribute

1. **Bug Reports**: If you find any issues with the reproduction scripts, please open an issue.
2. **Feature Suggestions**: We are looking for ways to improve the TFT model's performance on residuals.
3. **Optimizations**: Better hyperparameter tuning or alternative SARIMAX orders are welcome.

## Development Workflow

1. Fork the repository.
2. Create a new branch: `git checkout -b feature/your-feature-name`.
3. Make your changes and ensure `scripts/run_all.py` still runs successfully.
4. Push your changes and open a Pull Request.

## Coding Standards

- Follow PEP 8 for Python code.
- Ensure all new features are documented in the `README.md`.
- Keep the `data/` directory clean; use `scripts/download_data.py` for setup.

## Academic Integrity

This project is submitted for evaluation. While contributions are welcome for personal learning, please respect the academic guidelines for reproduction and derivative work.
